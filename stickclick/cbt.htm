   IT Training Packages.

The library is introducing several PCs running Windows NT 4.0 Workstation. These are to enable staff to become better aquainted with using Windows NT in advance of the University wide changeover happening this year.

There will be a PC at each campus and three at Frenchay (in User Services, Information Services and one in the LMT area for general use). These have 'Perception' training packages for the various MS Office applications (Word, Excel, PowerPoint and Access 97) and for the Windows NT user interface.
