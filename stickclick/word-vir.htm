  Introduction.

By far the greatest number of computer viruses around today are 'Word-macro' viruses. These are programs attached to Microsoft Word (or Excel) documents which run when you load the document into Word to view or edit them. Once infected, Word will go on to infect any other documents you open in it.

Most macro viruses are more annoying than harmful, though not all so. Fortunately there are two simple steps you can take to protect yourself from all known Word viruses...

  Prevention is better than cure, so make sure that:

  1. You have F-Secure 'Gatekeeper' switched on (see 'Gatekeeper' below).
  2. You don't double-click on any Word attachment icons  in Simeon.
    Instead, highlight (single click on the icon) and click the 'Save' button. 

    You can then open the attachment in the usual way from within Microsoft Word or from File Manager and it will automatically be checked by F-Secure Gatekeeper.

  Gatekeeper.

'Gatekeeper' is the part of F-Secure Anti-Virus that checks each document or program file as you use it.
It is switched on by default but can be switched on or off from the F-Secure Manager icon: 



  




    

  1. Minimise any open windows to find the icon on the lower left of the screen.
    2. 

  2. Click once to get the pop-up menu and select 'F-Secure Gatekeeper...'
    3. 

  3. In the 'F-Secure Gatekeeper Settings' dialog, check the 'Enable...' box,
or uncheck it to disable Gatekeeper.
    4. 




 

If you have an old PC (e.g. a 486) and you find that F-Secure slows down your 
  PC unacceptably, you can disable Gatekeeper, particularly if you don't have 
  MS Word installed. You will still get the automatic daily scan of your hard 
  disk. 

Otherwise, it is strongly recommended that you leave Gatekeeper enabled.

        What to do next.

  



1.    Disinfect the file

Gatekeeper will prevent you from opening an infected file but will not
automatically disinfect it. To do this you must run 'F-Secure Anti-Virus'.

    
  
     
  
  • Open the 'F-Secure Anti-Virus' program group in Program Manager.
    • 
  
    
  
    
     
  
    
  
    
  
  • If the infected file was on a floppy disk:
    • 
  
    
  -    Double-click the 'Scan drive A' icon. 
  
     
  
  • If the infected file was on your hard-disk:
    • 
  
    
  -    Double-click the 'Scan All Hard Drives' icon. 
  
     
  
  • After a few seconds you should see the 'Scanning...' box:
    • 
  
    
  
    
     
  
    
  
     
  
  • When the scan finds the infected document you will get a message that looks 
    similar to this:
    • 
  
    
  
    
     
  
    
  
     
  
  • Click on the 'Yes to All' button to disinfect this and any other infected 
    files it finds. This should leave the document intact and OK to open and edit 
    in Word.
    • 
  
     
  
  • Occasionally the scan will find a recent virus that it is not able to disinfect. 
    It then gives you the option of deleting the file or renaming it with a .VOC 
    extension (e.g.. 'filename.voc'). If this is an important file which you need 
    to work on, cancel out of the virus scanner, switch off the machine and try 
    booting up the PC from the 'Dr.Solomon's Magic Bullet' disk (If you work in 
    a UWE library, your IT co-ordinator should have a copy). This may be able 
    to deal with the odd virus which F-Secure cannot cope with.
    • 
  
     You can also use Word Viewer (a free Word document viewer from Microsoft, 
    available here) 
    to open an infected file, and then copy the text to a (clean) Word document. 
  
     Alternatively, contact library IT-Support (if you are library staff) or 
    the IT Services helpdesk. 





2.   Inform the sender.


If you received the document from someone else, either as an email attachment,
on a floppy or from a network drive, let them know.



    

  • If the document originated within UWE, they should have F-Secure installed,
so it is likely that they have 'Gatekeeper', turned off. Suggest they turn
'Gatekeeper' on and perform a scan of their files.
    • 





    

  • If you brought the document in from your home PC you should install an anti-virus program such as F-Secure. The UWE licence covers use by staff and students. Get in touch with library IT-Support (lib.sys@uwe.ac.uk) for a copy of
the F-Secure install disks.
    • 





    
  
  • Email attachments are probably the most likely way of receiving an infected 
    file. Simeon has a few peculiarities in it's integration with F-Secure:
    • 
  
     
  
      
    
    • Assuming Gatekeeper is enabled, it will NOT let you attach an infected 
      document to an email, but may delete the document without warning!
      • 
    
       
    
    • It WILL let you save an infected file to disk.
      • 
    
       
    
    • If you double-click on an infected Word attachment on a Windows 3.1 PC, 
      it will be loaded into Word and infect your copy. Only then will the document 
      be checked and discovered to be infected.
      
    
      • 
  
    



    
	
  • 
	Reply to the sender letting them know the problem, and suggest they contact
	anyone else they've sent the document to. If you have already disinfected
	the file you could always return it (with a different name to avoid confusion).






  
  
      So you think you've been infected...



The most common signs of a virus infection in Microsoft Word are:



    

    

  • F-Secure discovers an infection in 'NORMAL.DOT' (the standard Word template
file),
    • 

    

  • Only being able to save files as templates (.DOT files) rather than as
documents (.DOC files),
    • 

    

  • "Word Basic Error..." messages in Word,
    • 




F-Secure (with it's default setup) will scan the whole of your hard disk
once a day if you don't use your PC for more than 15 mins (you may well
come back from a tea-break to find it at work). If it finds an infected
copy of 'normal.dot' it will disinfect it but not always restore it completely.


A possible leftover of a virus infection is that the 'Macro...'
and 'Customize...' items have disappeared from the 'Tools' menu:




       
     
  



To fix this:

Find 'normal.dot' in File Manager (usually in the c:\msoffice\winword\template\ directory) and delete it. Word will create a new version the next time
it is run.

N.B. Doing this does mean you lose any settings you may have saved,
such as styles, default fonts, font sizes and paragraph formatting.
