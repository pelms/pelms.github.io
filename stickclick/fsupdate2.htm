Instructions for updating F-Secure version 4.02 for Windows 3.1 and Windows NT.

If you have a Hewlett Packard Vectra PC running Windows NT Workstation (not Thin Client), you will probably have a later version of F-Secure (version 4.03) and should click here to get the updater for that version.

Windows 3.1 PCs, Viglen PCs running Windows NT Workstation (not Thin Client) and the earliest of the HP Vectra PCs which went out, tend to run version 4.02 of F-Secure. These PCs have to be updated manually by copying over files to the F-Secure directory.

To check whether you have the right version, start F-Secure (in Windows NT by double-clicking the icon by the clock, in Windows 3.1 by double clicking the 'F-Secure Anti-Virus for Windows' icon).
Select 'About...' from the 'Help' menu and look at the version number in bold white text (e.g. 4.03.1151).

Instructions:

  1. Click here to download the updater for a Windows 3.1 PC.

    Click here to download the updater for a Windows NT PC.

  2. Choose 'Save this program to disk' (if asked) and save to a folder or the desktop.

  3. Find the program (update31 or updatent) in File Manager or NT Explorer and double-click it to run.



  4. Click the 'Unzip' button to copy over the new files.

  5. After the '..unzipped successfully' message click the 'Close' button to finish.

N.B. This page is updated at the start of each month (current version is for July 2000).
