Adobe Acrobat (.PDF) documents.

'Acrobat' files are self-contained documents which can contain searchable, formatted text and images. They can be viewed or printed using the free 'Acrobat Reader' viewer or directly from within a web browser if the Acrobat plugin is installed.

A useful feature of Acrobat documents is that Acrobat Reader will automatically scale them to fit the printer you are using - you don't need to worry whether the document will fit within the printer page margins.

Viewing an Acrobat document:

  • In Netscape, click on the link to the 'Acrobat' document. Netscape will launch Acrobat Reader and display the file. You can now print the document if there is a printer attached to your PC, but you cannot save the file to a disk from Acrobat Reader.

Saving to a disk:

  • Find the link to the 'Acrobat' document you want in your web browser.
  • Click the link with the Right-Hand mouse button to bring up a menu:
  • Select 'Save Link As...' (or 'Save Target As...') and then navigate to the a: drive to save to a floppy or the c: drive to save to the hard disk. The document will be saved as an Acrobat file with a .PDF extension.
  • You will need a PC with 'Acrobat Reader' installed to read or print the saved file.
