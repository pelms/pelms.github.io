Viewing Word attachments in Simeon.

Using 'Word Viewer' rather than 'MS-Word'.

N.B. 'Word Viewer' is a free program for viewing MS-Word documents. You can print from it, and cut and paste to another application (such as MS Word), but you cannot use it to create or edit documents.

Using Word Viewer with Simeon has several advantages:

  • You can't catch a virus...

    • By far the greatest number of computer virus around today are 'Word-macro viruses'. These are programs attached to Microsoft Word (or Excel) documents which run when you load the document into Word to view or edit them. Once infected, Word will go on to infect any other documents you use with it.
    If you double-click on an attached 'Word' document while in Simeon, it will normally launch MS-Word to view the file. Unfortunately, even if you have 'F-Secure' (or 'F-Prot') 'Gatekeeper' installed and running, from Simeon, it does not check the document for viruses before loading it into MS Word. This means you are not protected from virus infections.
    As Word Viewer only views the document, there is no chance of being infected with a virus.

  • You can view Word attachments on PCs that don't have MS-Office installed.

    • Since Word Viewer is installed on the shared network drive rather than on your Hard Disk.

  • You can read documents created in the latest versions of MS-Word (up to Word 97).

  • It is (a little) quicker to run.

Setting up Simeon to use 'Word Viewer'...

  1. From the Simeon menu, choose:  Tools > Options > Edit...  This will bring up the 'Options' box.
  2. Click on the 'Helper Applications' icon:  in the left-hand panel.
  3. Click in the box at the top containing 'Audio Aiff' (or possibly 'Acrobat Reader'), and select 'Microsoft Word 6'. 
    
    
    4. 


  4. 
In the 'View with:' box, change the entry, (usually from c:\msoffice\winword\winword.exe)
to: L:\wordview\wordview.exe  (at Bolland) or i:\wordview\wordview.exe
at most other sites, and click 'OK'.
    5. 


Now when you see a Word attachment in Simeon, instead of the usual Word icon  
you should see the Word Viewer icon.  

Double-click on it to view the attachment in Word Viewer.


If you need to edit the document. Save it to disk and then open it up
in Word in the normal way. F-Secure Gatekeeper will check the
file for viruses before loading it up.
