  Web pages (.HTM documents)

  • You can save a web page as either a plain text document or as an HTML document (containing all the web formatting tags).
  • To view or print an HTML document, you will need to load it into a Web browser (usually either Microsoft Internet Explorer or Netscape Navigator). To view or print a plain text file, you only need a text editor such as Windows Notepad or Write. If you are at Bolland library you can use the three Viewer/Printer PCs on level-3 to do this.

To save a web page as a plain text file:

  1. From the Netscape (or Internet Exxplorer) menu, choose   'File > Save As...'/LI>
  2. Select Plain Text in the 'Save File as Type' box.

  3. Make sure you name the file with a .txt extension
    (See Appendix: 'Netscape 'Save As...' Problems' ).

  4. Select 'a:' for the floppy-drive in the 'Drives' box and click 'OK'.

To save a web page as a HTML file:

  • Simply use 'File > Save As...' and accept the default setting of 'HTML Files' in the 'Save File as Type' box. The file will be saved with a .HTM extension.

  Images.

  • When you save a web page you usually don't save the images with it (see below). However, images can be downloaded seperately and can then be inserted into other applications such as MS Word (copyright permitting).

To save an image from a web page:

  • Click the image with the Right-Hand mouse button to bring up a menu:
  • Select 'Save Image As...' (or 'Save Picture As...') and then navigate to the a: drive to save to a floppy or the c: drive to save to the hard disk.
  • The image will usually be either a JPEG file (with a .JPG file extension eg. image.jpg) for full colour images or photographs, or a GIF file (with a .GIF extension eg. image.gif) for graphics and artwork.
  • These files can be inserted into a MS Word document using:
      'Insert > Picture > From File...' from the Word menu.

  Web pages with images.

  • If you save a web page using 'Save As...' from the browser menu, you will only get the text of the page and not the images (unless you are using Microsoft Internet Explorer 5.0 and choose 'Web Page, Complete' in the 'Save as type' box).
  • If you are using Netscape Navigator Gold 3.0 (which is the standard on library staff PCs in the 'LAN Workplace' Program Group and on the library student network PCS), there is a work around however...

To save a web page complete with it's images:

  1. From the Netscape Navigator menu choose:   'Edit Document'
  2. A box titled 'Save remote Document' should pop up.
  3. Make sure the 'Save images with document' box is ticked and click 'Save'.
  4. Read the copyright warning if it appears.
  5. In the 'Save As' box, navigate to the a: drive to save to a floppy or the c: drive to save to your hard disk and click 'OK'.
  6. The web page will be saved along with all it's images.
  7. To view the saved page, open it up in your web browser using:
    'File > Open file in Browser...' (or 'Open') from the menu.

  Acrobat (.PDF) documents

  • 'Acrobat' files are self-contained documents which can contain searchable, formatted text and images. They can be viewed or printed using the free 'Acrobat Reader' viewer or directly from within a web browser if the Acrobat plugin is installed.

    A useful feature of Acrobat documents is that Acrobat Reader will automatically scale them to fit the printer you are using - you don't need to worry whether the document will fit within the printer page margins.

Viewing an Acrobat document:

  • In Netscape, click on the link to the 'Acrobat' document. Netscape will launch Acrobat Reader and display the file. You can now print the document if there is a printer attached to your PC, but you cannot save the file to a disk from the Reader.

Saving to a disk:

  • Find the link to the 'Acrobat' document you want in your web browser.
  • Click the link with the Right-Hand mouse button to bring up a menu:
  • Select 'Save Link As...' (or 'Save Target As...') and then navigate to the a: drive to save to a floppy or the c: drive to save to the hard disk. The document will be saved as an Acrobat file with a .PDF extension.
  • You will need a PC with 'Acrobat Reader' installed to read or print the saved file.
