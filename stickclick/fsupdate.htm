   F-Secure 4.03 July update.

Protect yourself from the 'ILOVEYOU' LoveLetter virus!

If you have a Hewlett Packard Vectra PC running Windows NT Workstation (not Thin Client), you can download the virus checker updater below.

This program will only update F-Secure versions 4.03 and above, i.e. not the version on Windows 3.1 machines or on some older NT machines.
If you have an earlier version of F-Secure click here.

To check which version you have, start F-Secure by double-clicking the icon by the clock, select 'About...' from the 'Help' menu and look at the version number in bold white text (e.g. 4.03.1151).

Instructions:

  1. Click here to download the updater.

  2. Choose 'Save this program to disk' (if asked) and save to a folder or the desktop.

  3. Find the program (fsupdate) in NT Explorer and double-click it to run.

After a few seconds the update will be complete and you should be protected from the latest viruses.

N.B. This page is updated at the start of each month (current version is for July 2000).
